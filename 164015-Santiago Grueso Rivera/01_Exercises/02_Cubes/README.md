# Sum of all Cubes

Sum of all cubes of every integer from number A to number B.

## Problem Definition

Write a Python program that calculates the sum of the cubes of every integer from A to B (inclusive)

### Input

    1
    3

### Output

    36

## Use and test

    cd 02_Cubes
    python3 cubes.py
    cd tests
    python3 cubes_test.py
